# ADR-007: Move the Deploy Window (ACCEPTED, 2024-01)

## Status

Accepted. Supersedes the deploy-day decision in ADR-001.

## Context

Friday deploys were causing weekend pages. The on-call rotation revolted. A
strongly-worded Slack thread ensued. Management noticed.

## Decision

Deploys moved to Wednesday, mid-week, so a bad deploy has business days to bake
before the weekend.

Approval still goes through the existing owner; this ADR is only about the day.

## Notes

This does not change the release cadence, only the deploy day. (Narrator: a later
runbook would change it again.)
